# Exploring-and-Visualizing-a-Simple-Dataset
# 🌸 Iris Dataset — Exploratory Data Analysis (EDA)

## 📌 Objective
Load, inspect, and visualize the Iris dataset to understand data trends and distributions.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Visualizations
- Scatter Plot — species clustering
- Histograms — feature distributions
- Box Plots — outlier detection
- Pair Plot — all features at once

## 🚀 How to Run
```bash
pip install pandas matplotlib seaborn scikit-learn
python iris_eda.py
```

## 💡 Key Insight
Setosa is clearly separable from Versicolor and Virginica using petal measurements alone.
